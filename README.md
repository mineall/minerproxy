# 一键安装升级命令

```bash
/bin/bash -c "$(curl -fsSL https://raw.githubusercontent.com/mineall/minerProxy/main/ssminer.sh)"
```

> 选择 `1`  安装程序；

> 选择 `2`  更新程序；

> 选择 `3`  查看服务状态；

> 选择 `4`  卸载程序。
